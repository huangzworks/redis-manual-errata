# 《Redis使用手册》勘误

本项目用于收集[《Redis使用手册》](http://redisguide.com)一书的勘误信息，读者朋友也可以通过 PR 提交自己发现的错误。

在提交错误之前，请先确认手上书本的印刷版次，并检查你发现的错误在当前版次或者后续版次是否已被修复。  
如果你对此感到不确定的话，那么可以将你发现的错误发布到 issue 页面，然后由书本作者代为检查。

非常感谢你为提高本书后续印刷版本所做的努力！
